Callbacks

public protocol Callbacks

Provides an interface for programs to respond to callbacks from the bgfx library

  • reportError(_:message:)

    Called when an error occurs in the library

    Remark

    If the error type is not ErrorType.DebugCheck, bgfx is in an unrecoverable state and the application should terminate

    This method can be called from any thread

    Declaration

    Swift

    func reportError(_ type: BgfxError, message: String)

    Parameters

    type

    The type of error that occurred
    message

    Message string detailing what went wrong
